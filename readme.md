<p><a target="_blank" rel="noopener noreferrer" href="/dhyan1999/Helmet-Detection--Computer-Vision-/blob/main/img/step3.png"><img src="/dhyan1999/Helmet-Detection--Computer-Vision-/raw/main/img/step3.png" alt="step3" style="max-width:100%;"></a></p>

</ul>
<p><a target="_blank" rel="noopener noreferrer" href="/dhyan1999/Helmet-Detection--Computer-Vision-/blob/main/img/step41.png"><img src="/dhyan1999/Helmet-Detection--Computer-Vision-/raw/main/img/step41.png" alt="step41" style="max-width:100%;"></a></p>

<ul>
<li>Step5 - Now after saving the picture, a class file and a txt file is formed in the folder. Class file contains the 2 classes (helmet and no helmet) that we made and the txt file contains the coordinates of the rectangle that we made around the person’s face.</li>
</ul>
<p><a target="_blank" rel="noopener noreferrer" href="/dhyan1999/Helmet-Detection--Computer-Vision-/blob/main/img/step5.png"><img src="/dhyan1999/Helmet-Detection--Computer-Vision-/raw/main/img/step5.png" alt="step5" style="max-width:100%;"></a></p>

</li>
<li>
<p>Step7 - So, finally after saving the changes made to the last picture one can see 1 class file and different txt files in the folder.</p>
</li>

<p>Step8 - We can combine all these txt files into one xml file and we can use that xml file in the code to detect the person wearing helmet.</p>
</li>
</ul>
